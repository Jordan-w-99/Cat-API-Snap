Description:

For this task, my approach was to make a simple game based on the card game 'Concentration', but instead of cards, use randomly selected cat pictures from a fun API I found called: 'The Cat API'.
The API itself returns JSON data holding information about each cat fetched such as breed, image categories, image URL, and various other data (I only really need the URL).
When making the request I specify the amount of images needed, as well as the desired image file type and size.
Each time a new round is loaded, it uses the Javascript Fetch API to get the required amount of images and presents them as face-down cards.
The player then has to click two cards to find a pair, if they are unsuccessful the cards turn back over.
Once all the cards have been paired, you win!
The player can choose different grid sizes from the dropdown under the game, and the cards will scale accordingly to fill the canvas area.

Technologies
used:

  • HTML, CSS, Javascript.
  • P5.js Javascript library.
  • Javascript Fetch API.
  • The Cat API.
